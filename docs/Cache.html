Interface: Cache

Cache

A cache is responsible for storing and retrieving tapes and ops from a cache, and if necessary instructing an adapter to fetch items from a data source. A cache must implement all of the following callbacks: * `fetchTx()` * `fetchTxBy()` * `fetchOps()`
